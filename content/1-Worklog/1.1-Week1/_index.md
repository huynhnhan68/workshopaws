### Week 1 Objectives:

* Research an overview of AWS Cloud and the **Pay-as-you-go** pricing model.
* Register and set up an **AWS Free Tier** account for hands-on practice.

* Get familiar with **Markdown**, **Hugo**, **Snagit** tools and AWS resource optimization methods.
* Analyze basic AWS services and cost structures for the final project workshop.
